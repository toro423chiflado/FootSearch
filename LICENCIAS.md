# Licencias de club (IDs de un solo uso)

Estas son **100 licencias fijas** incluidas en el sistema. Cada una sirve
para registrar **un** club; al usarse queda marcada y no se puede reutilizar.
El sistema genera además ~900 aleatorias hasta completar 1000 disponibles.

Para listar las libres en cualquier momento (endpoint admin):

    curl "http://localhost:4000/api/admin/licencias?estado=libres&limit=100" \
      -H "x-admin-key: TU_CLAVE_ADMIN"

> Nota: en el seed, las 2 primeras licencias se usan para crear los clubes
> demo (Atlético Cóndores FC y Deportivo Manglar), así que esas dos aparecerán
> como usadas. Las otras 98 (y las aleatorias) están libres.
