# Base de datos de DNIs habilitados — FootSearch

Este archivo lista los **200 DNIs** cargados en la base de datos local (tabla `dni_habilitados`). Solo con uno de estos documentos se puede registrar una cuenta de **Jugador** o **Cazatalentos**. Cada DNI es de **un solo uso**: al registrarse queda marcado como usado y ya no sirve para otra cuenta. (Los Clubes NO usan DNI; usan licencias `FS-XXXX-...`.)

## Cómo se usa

1. Al registrarte como jugador o cazatalentos, ingresa un DNI de la lista **DNIs LIBRES**.
2. El formato es de **8 dígitos** (ej. `11111111`).
3. El backend lo verifica contra esta base. Si no existe o ya fue usado, el registro se rechaza.

## Consultar el estado desde el panel de administrador (Postman)

```
GET  http://localhost:4000/api/admin/dnis?estado=libres
Header:  x-admin-key: <tu ADMIN_KEY>
```

Valores de `estado`: `libres`, `usadas`, `todas`.
